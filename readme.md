1.	Make a folder with your "**PROJECT-NAME**".
2.	Run the setup script.
    Open PowerShell and run:
        C:\Inloops-clients\new_project.ps1 -RepoName "**PROJECT-NAME**"
3.	Open the project in Antigravity.
    Inside Antigravity (under requirements folder):
        – Set up the knowledge base
        – Set up the design system
        – Set up the sitemap
4.	Install the project tools.
    In the terminal, run:
    npm install
5.	Use the Agent panel to manage pages.
    Available commands:
        – /create → sets up all needed files
        – /create-page [page-name].md → makes a new page and its content file
        – /modify-page [page-name].md → updates a page and its content file
        – /add-page [page-name].md → adds a page with its content file
6.  To run
        npm run dev
