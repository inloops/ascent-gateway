# Design System

Visual language for Ascent Travels.

## Brand Identity

- **Brand Name**: Ascent Travels
- **Tone of Voice**: Bold, Empowering, Rugged, trustworthy.
- **Visual Style**: High-Contrast, Clean, Dynamic.

## Colors

| Usage      | Color Name   | Hex Code | Description                      |
| :--------- | :----------- | :------- | :------------------------------- |
| Primary    | Alpine Blue  | #0F172A  | Deep navy/slate, headers, footer |
| Secondary  | Summit Gold  | #F59E0B  | CTAs, Important buttons, Alerts  |
| Background | Snow White   | #F8FAFC  | Main page background             |
| Surface    | Glacier Grey | #FFFFFF  | Cards, Modals                    |
| Text       | Granite      | #334155  | Body text                        |
| Accent     | Forest       | #10B981  | Success states, eco-badges       |

## Typography

- **Font Family (Headings)**: 'Oswald', sans-serif - _Tall, strong, condensed font implies height and strength._
- **Font Family (Body)**: 'Inter', sans-serif - _Clean, modern, and highly legible._
- **Base Size**: 16px

## Layout

- **Container Width**: 1280px (max-w-7xl)
- **Border Radius**: 4px (rounded-sm) - _Sharper corners for a rugged, precise feel._
- **Spacing**: Comfortable, but not too loose.

## Animations (GSAP)

- **Entrance**: `anim-fade-up` (Strong vertical movement).
- **Scroll**: Parallax effects on mountain background images.
- **Hover**: Buttons elevate slightly; Cards reveal details on hover.
